# Veejay_PoetryWeb 🖤📖

**Veejay_PoetryWeb** is a dark, cinematic poetry web experience — part digital journal, part gallery, part creative tool.  
It is designed for writers who think deeply, feel intensely, and want their words to live beyond plain text.

This project is built with ** HTML, CSS, and JavaScript**, deployed via **GitHub Pages**, and developed locally using VS Code.


## ✨ Features

- 📝 **Poetry Editor**
  - Live preview while writing
  - Auto-saving drafts (local browser storage)
  - Submit poems to a personal library (device-based)

- 📚 **Poetry Library**
  - Displays submitted poems + built-in seed poems
  - Search by title or content
  - Filter by tags

- 🖼️ **Export as Image**
  - Social-media–ready formats:
    - Square (1:1)
    - Portrait (4:5)
    - Story (9:16)
  - Smart auto-fit font sizing (adapts to poem length)
  - One-click PNG export

- 🎨 **Dark Cinematic Theme**
  - Mood toggle
  - Grain, gradients, and minimalist UI
  - Reader-first, distraction-free design

## 🔒 Data & Storage Notes

- Poems are stored using **localStorage**
- Submissions are saved **per device/browser**
- There is **no backend** (by design)

This keeps the project:
- Lightweight
- Private
- Easy to deploy on GitHub Pages

Future versions may introduce optional cloud storage.
